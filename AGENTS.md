# Versus Cypher agent guide

**[`MISSION.md`](./MISSION.md)** - ethos, future bet, hard no's, and north-star checks.

For how the system is built, read:

**[`docs/ARCHITECTURE.md`](./docs/ARCHITECTURE.md)**

**[`ROLLING.md`](./ROLLING.md)** - living design, settled principles, and unresolved questions. Do not silently promote an open question into protocol truth.
